## Recent Updates

Stay informed about new features, improvements, and fixes in jls. You receive notifications for breaking changes and major releases directly in your dashboard.

- Git sync for version-controlled content updates.

## Upgrade Guide

Follow these steps to upgrade your jls instance to the latest version. Back up your data first.

<Callout kind="alert">
Always test upgrades in a staging environment before production deployment.
</Callout>

<Steps>
  <Step title="Backup Data" icon="database">
    Export your documentation spaces:

````bash
jls export --space=all --output=backup-$(date +%Y%m%d).tar.gz
````
  </Step>
  <Step title="Update CLI" icon="download">
    Install the latest CLI version using your package manager.

<CodeGroup tabs="npm,yarn">
````bash
npm install -g @jls/cli@latest
````
````bash
yarn global add @jls/cli@latest
````
</CodeGroup>
  </Step>
  <Step title="Run Migration" icon="settings">
    Apply database migrations:

````bash
jls migrate up --version=1.2.0
````
  </Step>
  <Step title="Verify" icon="check-circle">
    Restart services and check status:

````bash
jls status
jls health-check
````
  </Step>
</Steps>
