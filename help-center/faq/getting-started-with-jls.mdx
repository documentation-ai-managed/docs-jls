## Overview

jls lets you organize project docs easily.

## Create a Workspace

1. Go to `https://app.jls.io`
2. Click **New Workspace**
3. Name it for your project
4. Invite team members

## Create Documents

- Use the **New Document** button
- Organize in folders

Your workspace is ready!
